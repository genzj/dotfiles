# zim-my-config

## Usage

This ZIM module is a part of my chezmoi config. Check [its
README](https://github.com/genzj/dotfiles) to configure it.

## Features

1. Setup [fzf](https://github.com/junegunn/fzf?tab=readme-ov-file) with
[fd](https://github.com/sharkdp/fd) and [bat](https://github.com/sharkdp/bat)
supports

## Aliases

### Branch

  * `Gl1` displays the last log
  * `Gl1p` displays the last log and patch
  * `Gl1s` displays the last log and file stats

### Merge

  * `Gmff` ff-only merge

### Main working tree

  * `Gw` displays the working tree status

### Branch

  * `GbS` displays all branches briefly
  * `GbU` set upstream of current branch to the default branch
